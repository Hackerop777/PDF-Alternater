# 📄 PDF Alternater: Automated Journal Maker

A lightweight, blazing-fast web tool designed to automate the tedious process of formatting college practical journals. 

Instead of manually merging PDFs, this tool instantly interleaves a **1-page journal cover/template** before every page of your **multi-page assignment**, generating a perfectly formatted final PDF in seconds.

## ✨ Features
- **⚡ 3-Click Workflow:** Upload template, upload assignment, and generate. That's it!
- **🔒 100% Private & Secure:** All PDF processing happens locally in your browser using `pdf-lib`. Your files are **never** uploaded to any server.
- **🚀 No Installation Required:** Works directly in any modern web browser. No Python, Node.js, or backend setup needed.
- **🎓 Perfect for Students:** Specifically built to solve the hassle of formatting practical journals, lab records, and assignments.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Core Library:** [pdf-lib](https://pdf-lib.js.org/) (for client-side PDF manipulation)

## 🚀 How to Use
1. Open `index.html` in your web browser (or visit the hosted GitHub Pages link).
2. **Step 1:** Upload your 1-page Journal Template (e.g., `practical_paper.pdf`).
3. **Step 2:** Upload your Multi-page Assignment (e.g., `assignment.pdf`).
4. **Step 3:** Click **Generate Final Journal**.
5. The perfectly interleaved PDF will automatically download to your device!
